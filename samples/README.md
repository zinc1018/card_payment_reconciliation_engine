# Samples

This folder is reserved for sanitized sample data only.

Do not place real exports from:

- Versapay
- Global Payments
- production reconciliation workbooks

Recommended sample strategy:

- header-only CSV examples for each supported layout
- tiny invented datasets with:
  - Visa, MC, and Amex rows
  - leading-zero authorization codes
  - one Global Payments batch that maps to multiple Versapay batch ids
